It is almost always required to make some plots if you are presenting data. In this article, we will explore a few ways to make your plots production quality and integrate it with your articles. Among the multiple ways of generating plots, we will explore three

  1. using python-matplotlib
  2. using gnuplot
  3. using pgfplots package

Plots using python-matplotlib

I am using python and matplotlib combo because I think this might be the most widely used combo for plotting. There exists packages like bokeh and seaborn which are amazing at what they do. However, matplotlib is no less and suffices for most of the use cases you will need for plotting 2D or 3D plots. I assume that you already know how to plot data using matplotlib but here only to know how to beautify it and make it better. So lets explore it.
